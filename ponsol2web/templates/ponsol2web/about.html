Start to predict

To start using PON-Sol2, you just need to provide protein FASTA sequences and amino acid substitutions. For protein FASTA sequences, there are 2 input ways: directly input FASTA sequences or input protein IDs(GI, Ensemble ID or UniProt). For both ways, you can directly provide the corresponding content or upload files, and can submit multiple queries simultaneously.

If you just want to get the input example, please click the "Example" text on the input page.

Input FASTA sequences

If you have the complete FASTA sequences, you can input FASTA sequences. FASTA sequences and amino acid substitutions must be provided, and email is optional. If email is provided, the results will be sent to you by email when ready.

  1. For FASTA sequences, each sequence should have a header line starting with greater than sign (>) followed by amino acids sequence. Amino acids sequence needs to start from a new line, and can be one-line or multiline.
  2. For amino acid substitutions, substitutions corresponding to a sequence should contain the same header line as the sequence. An amino acid substitution consists of three parts: origin amino acid, position and new amino acid. For example, "A2M" means the second amino acid A (Alanine) is substituted by M (Substitute). Each protein sequence can correspond to multiple amino acid substitutions, and amino acid substitutions in the same FASTA sequence should be on the different lines.

Input Protein IDs

If you just have the protein IDs, PON-Sol2 will find the FASTA sequences according to the IDs. Protein IDs, amino acid substitutions and type of IDs must be provided, and email is optional. If email is provided, the results will be sent to you by email when ready.

As protein IDs are usually short, there is only one input box. The IDs should be preceded by greater than sign (>). You can provide multiple amino acid substitutions, and the amino acid substitutions of the same protein need to put to the different lines after the line of ID. The following example means the protein that ID is Q5VT03 has two amino acid substitutions, the first is the first amino acid from M (Methionine) to A (Alanine) and the second is the third amino acid from F (Phenylalanine) to Q (Glutamic acid). 

>Q5VT03
M1A
F2Q

Predict result

PON-Sol2 provides complete result presentation pages. For each submission you make we call it a "Task" and for each amino acid substitution we call it a "Record". Each task and record has its own detail page. So there are 3 pages of predict result:

  1. Task list. Click "Result" of navbar, you will get a task list according to your IP.
  2. Task detail and record list. Click status of task, you will get task detail containing status of task, start time, end time, email and record list.
  3. Record detail. Click protein name of record, you will get record detail containing protein name, FASTA sequence, amino acid substitution, result and status.

If you provide email when predicting, the results will be sent to you by email when ready.

Dataset

For PON-Sol2, altogether 11201 amino acid substitutions in 77 proteins are collected. After filtering, 5666 variants (2798 solubility decreasing, 1929 increasing and 939 without effect on solubility) are used for training, and 662 variants (338 solubility decreasing, 237 increasing and 87 without effect on solubility) are used for blind testing.  download
